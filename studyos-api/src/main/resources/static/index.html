Perguntas frequentes

Ficou com alguma dúvida?

O StudyOS é gratuito?+
Sim. Você pode criar sua conta e usar todas as funcionalidades principais sem precisar pagar nada ou cadastrar cartão de crédito.
Preciso saber qual método de estudo usar?+
Não. O StudyOS apresenta os 12 métodos com descrição e dicas de uso você escolhe o que combina com seu momento. Pode alternar entre eles a qualquer hora.
Como funciona o sistema de XP e níveis?+
Cada sessão concluída gera XP com base no tempo e método usado. Acumulando XP você sobe de nível são 6 no total, do Iniciante ao Lendário. Cada nível desbloqueia novas conquistas.
O que é repetição espaçada (SM-2)?+
É um algoritmo que agenda revisões no momento ideal para sua memória não muito cedo (quando você ainda lembra bem) nem tarde demais (quando já esqueceu). O StudyOS aplica isso automaticamente nos seus flashcards.
Funciona no celular?+
Sim, o StudyOS é totalmente responsivo. A experiência funciona bem em desktop, notebook e celular sem precisar instalar nenhum aplicativo.
Meus dados estão seguros?+
Sim. Usamos autenticação JWT e suas senhas são armazenadas com criptografia BCrypt nunca em texto puro. Seus dados de estudo ficam associados apenas à sua conta.
Posso usar para qualquer matéria?+
Sim. O StudyOS é genérico você cria as matérias que quiser com emoji, cor e meta semanal personalizada. Funciona para vestibular, concursos, faculdade, idiomas ou qualquer coisa que você esteja estudando.
